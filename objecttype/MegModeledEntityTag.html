MegModeledEntityTag

Prefix:megmodeledentity@Identity Format:Identified using the format megmodeledentity@<entity>.Description:A modeled entity doesn't necessarily have to have a model, and it can have multiple models.View source

Tags​

Tag: MegModeledEntityTag.body_clamp_uneven​

Returns:ElementTag(Boolean)Mechanism:body_clamp_unevenDescription:Returns whether the body clamp is uneven on the modeled entity. Body clamp is used to clamp the body when the entity is standing still. See also: MegModeledEntity.max_body_angleSee also: MegModeledEntity.min_body_angle

Tag: MegModeledEntityTag.entity​

Returns:EntityTagDescription:Returns the underlying entity of the modeled entity.

Tag: MegModeledEntityTag.entity_visible​

Returns:ElementTag(Boolean)Mechanism:entity_visibleDescription:Returns whether the base entity of the modeled entity is visible.

Tag: MegModeledEntityTag.head_clamp_uneven​

Returns:ElementTag(Boolean)Mechanism:head_clamp_unevenDescription:Returns whether the head clamp is uneven on the modeled entity. Head clamp is used to clamp the head when the entity is moving. See also: MegModeledEntity.max_body_angleSee also: MegModeledEntity.min_body_angle

Tag: MegModeledEntityTag.max_body_angle​

Returns:ElementTag(Number)Mechanism:max_body_angleDescription:Returns a ElementTag(Number) of the current maximum body angle of this entity. See also MegModeledEntity.min_body_angle

Tag: MegModeledEntityTag.max_head_angle​

Returns:ElementTag(Number)Mechanism:max_head_angleDescription:Returns a ElementTag(Number) of the current maximum head angle of this entity. See also MegModeledEntity.min_head_angle

Tag: MegModeledEntityTag.min_body_angle​

Returns:ElementTag(Number)Mechanism:min_body_angleDescription:Returns a ElementTag(Number) of the current minimum body angle of this entity. See also MegModeledEntity.max_head_angle

Tag: MegModeledEntityTag.min_head_angle​

Returns:ElementTag(Number)Mechanism:min_head_angleDescription:Returns a ElementTag(Number) of the current minimum head angle of this entity. See also MegModeledEntity.max_head_angle

Tag: MegModeledEntityTag.model[(<model>)]​

Returns:MegActiveModelTagDescription:Returns the active model with the specified name on the modeled entity. If no name is specified, returns the first model on the modeled entity.

Tag: MegModeledEntityTag.models​

Returns:MapTag(MegActiveModelTag)Description:Returns a map of all models on the modeled entity. The keys are the names of the models, and the values are the active models.

Tag: MegModeledEntityTag.player_mode​

Returns:ElementTag(Boolean)Mechanism:player_modeDescription:Returns a ElementTag(Boolean) of whether the body will rotate like a player (i.e. will not try to match the head angle).

Tag: MegModeledEntityTag.rotation_delay​

Returns:DurationTagMechanism:rotation_delayDescription:Returns a DurationTag of the delay before the body auto rotates to match the head's rotation. See also: MegModeledEntity.rotation_duration

Tag: MegModeledEntityTag.rotation_duration​

Returns:DurationTagMechanism:rotation_durationDescription:Returns a DurationTag the body takes to auto rotate back to the head's rotation. See also: MegModeledEntity.rotation_delay

Tag: MegModeledEntityTag.should_save​

Returns:ElementTag(Boolean)Mechanism:should_saveDescription:Returns whether the modeled entity should be saved on unload.

Tag: MegModeledEntityTag.stable_angle​

Returns:ElementTag(Number)Mechanism:stable_angleDescription:Returns the angle in degrees where the body rotation can exceed the clamp before snapping towards the clamped values.
