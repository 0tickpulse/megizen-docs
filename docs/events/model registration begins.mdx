#
```js
meg model registration begins
```
This event is fired when the model generator enters and exits a phase, and when the generator has finished everything.

## Switches

| Switch          | Description                                             |
| --------------- | ------------------------------------------------------- |
| `phase:<phase>` | Only processes the event if a specific phase is active. |

## Contexts

| Context           | Description                                                                                                                                                     |
| ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `<context.phase>` | returns an ElementTag of the current model registration phase. Can be PRE_IMPORT, POST_IMPORT, PRE_ASSETS, POST_ASSETS, PRE_ZIPPING, POST_ZIPPING, or FINISHED. |
